Samourai Wallet


Samourai is a Bitcoin wallet for Android with sophisticated built-in privacy. The primary feature is its integration of the Whirlpool protocol, which aims to improve privacy and thwart chain-analysis. Samourai also includes many other features such as "Ricochet" to improve fungibility, "Stonewall" for better transaction privacy, Tor to mask IPs, as well as Taproot, PayNyms, and more.

Sparrow Wallet


Sparrow is a desktop wallet with an emphasis on security, privacy, and usability. Similarly to Samourai, Whirlpool & Stonewall can be used in order to obfuscate transaction history and improve privacy. Meticulous organization and micromanagement of funds is made optionally available for advanced users, while still providing an accessible interface for beginners. Sparrow also features a private block explorer, Tor integration, support for PayNyms, and several methods of connection for interacting with the Bitcoin network.

Blue Wallet


Blue Wallet is a user-friendly wallet for Android, iOS, & macOS. Several wallets can be set up and utilized at once, including regular, lightning, multisig, and view-only wallets. Payjoin can optionally be used to improve transaction privacy. Blue Wallet also allows the user to set up a "fake" wallet, which acts as a decoy and provides plausible deniability. While the lightning network is supported, lightning wallets are custodial; if it chooses to, the company is able to steal, monitor, or otherwise mishandle your lightning funds however it wants. However, this removes the need to set up your own payment channels, and allows you to receive lightning payments while offline.

Wasabi Wallet


Wasabi Wallet is a relatively popular and supposedly privacy-focused desktop wallet. Wasabi is not recommended; use Sparrow Wallet instead. Historically, Wasabi has used a very flimsy privacy technique, which has led to the deanonymization of countless unsuspecting users. Wasabi has refused to acknowledge the deanonymization caused by their lousy and primitive implementation. Additionally, their implementation of Coinjoin is only usable with large amounts, and the coordinator is now disallowing certain funds to be mixed.
